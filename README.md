# Netauto

Netauto sera una capa de abstraccion que funcione para cualquier dispositivo de red sin importar su modelo, la idea es proveer una herramienta en la que se indique desde como conectarse a un dispositivo, que comando debe ejecutar y hasta como procesar su salida sin mebargo para el usuario sera inherente las configuraciones, el solo debera pedir lo que necesite, Ejemplo:

FROM 192.X.X.X GET ipaddress -> OUT -> JSON
<br>
FROM 100.X.X.X GET interface -> OUT -> CSV

Nota: De momento solo es una idea en nacimiento, no se tiene claro si sera de dicha forma.
